## Deployment

This project is deployed using GitHub Pages.

Steps:

1. Push project to GitHub.
2. Open Repository Settings.
3. Navigate to Pages.
4. Select:

   * Branch: main
   * Folder: /root
5. Save.
6. GitHub generates the live website URL.
